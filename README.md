# vektor.fm — missions for Claude Code

Hands-on missions that make you genuinely good at Claude Code — verified
in your own terminal by your own agent, not self-reported.

## Install (two lines, inside Claude Code)

```
/plugin marketplace add julianlaycock/vektor-plugin
/plugin install vektor@vektor
```

Then start your first mission:

```
/vektor:start
```

## What you get

- **Mission 01 — Token Discipline.** Cut a 2,000-line read to a grep:
  make Claude search before reading, cap every large read, and delegate
  breadth to subagents. ~25 minutes, in a real repo of yours.
- **/vektor:check** — a strict examiner that audits your setup and your
  agent's live behavior, then prints a scorecard. It takes nobody's word
  for anything. Clear all four criteria and you're CLEARED.
- **/vektor:progress** — where you stand, any time.

Progress lives in a local file (`~/.vektor/progress.json`). No account,
no server, nothing leaves your machine.

New missions are announced at [vektor.fm](https://vektor.fm).

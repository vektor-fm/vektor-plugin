# vektor.fm

Hands-on missions that make you genuinely good at Claude Code —
verified in your own terminal, not self-reported.

## Install

Inside Claude Code:

```
/plugin marketplace add <public-repo-tbd>
/plugin install vektor@vektor
```

## Use

| Command | What it does |
|---|---|
| `/vektor:start` | Start (or resume) your current mission |
| `/vektor:check` | Run the strict examiner — earn your CLEARED scorecard |
| `/vektor:progress` | See your mission tree |

Mission 01 — **Token Discipline** (~25 min): stop your agent burning
money reading 2,000-line files. Three habits, installed permanently,
proven live.

Progress is stored locally in `~/.vektor/progress.json`. Nothing phones
home.
